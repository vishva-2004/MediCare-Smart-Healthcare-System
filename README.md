# MediCare - Smart Healthcare System

A full stack web application for managing hospital operations. The system helps patients book appointments, doctors manage their schedules, and administrators handle hospital activities through a simple and secure interface. One of the key features is the **AI Symptom Checker**, which analyzes patient symptoms and suggests the most suitable medical department and doctor specialization before booking an appointment.

## Features

* AI Symptom Checker
* Secure user login
* Role based access
* Patient management
* Doctor management
* Appointment booking
* Department management
* Medical record management
* Search doctors by department
* Input validation
